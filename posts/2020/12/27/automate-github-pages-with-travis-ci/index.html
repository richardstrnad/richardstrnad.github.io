Automate GitHub Pages with Travis CI

To have a CICD pipeline for my github pages site, i use the following travis config. GITHUB_AUTH_SECRET is set in the settings page of the respective travis project.

---
install:
  - curl -LO https://github.com/gohugoio/hugo/releases/download/v${HUGO_RELEASE}/hugo_extended_${HUGO_RELEASE}_Linux-64bit.deb
  - sudo dpkg -i hugo*.deb

script:
  - hugo

deploy:
  - provider: pages
    local_dir: public
    repo: richardstrnad/richardstrnad.github.io
    target_branch: master
    skip_cleanup: true
    github_token: $GITHUB_AUTH_SECRET
    keep_history: true
    on:
      branch: master

env:
  global:
    - HUGO_RELEASE=0.79.1
